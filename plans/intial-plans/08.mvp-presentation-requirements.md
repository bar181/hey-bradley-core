## 3. Chat & Listen Mode Enhancements

### Dropdown Example Selector

Both Chat and Listen modes need a dropdown (or card grid) to select which demo to run:
- Shows all available examples (10-12)
- Each entry shows name + brief description + theme
- Selecting triggers the canned demo simulation

### Progressive Build Sequences (5-60 seconds)

Each demo should show a site building from blank:
1. **[0s]** Blank canvas
2. **[2s]** Theme/palette applies
3. **[5s]** Hero section appears
4. **[8s]** First content section
5. **[12s]** Social proof (quotes/logos)
6. **[16s]** More sections build
7. **[20-60s]** Full site complete

Short demos (5-10s): 3-4 sections
Medium demos (15-25s): 5-6 sections
Full demos (30-60s): 7-8+ sections

### Chat Changes Step-by-Step

After initial build, the demo should show **modification commands**:
- "make it dark" → theme switches
- "add pricing" → pricing section appears
- "headline: Launch Your Product" → hero updates
- "remove the FAQ" → section disappears

## 4. Image Section Enhancements

### Image Selector & Config

Add advanced image configuration options to the image section editor:
- **Image source selector** — curated library, URL paste
- **Gradient overlay** — configurable gradient direction + colors
- **Ken Burns effect** — slow zoom/pan animation on static images
- **Parallax** — scroll-linked movement
- **Zoom on hover** — scale 1.0 → 1.05 on mouse enter
- **Overlay text** — heading + subheading positioned over the image
- **Aspect ratio** — 16:9, 4:3, 1:1, free
- **Object fit** — cover, contain, fill

These options should be accessible in the right panel image editor with toggle switches (shadcn Switch) for each effect.

## 5. Theme Lock Toggle (Pre-MVP)

### Concept

A toggle in the theme editor panel: **"Lock Theme"**

When locked:
- Theme preset, palette, font, mode, and spacing are frozen
- Section-level style changes are blocked
- Only content (text, images, component toggles) can be edited
- Chat/Listen commands that would change theme are rejected with "Theme is locked"
- Visual indicator: small lock icon in TopBar next to theme name

When unlocked (default):
- Everything editable as normal

### Purpose

Prevents accidental theme changes during content editing. Important for the demo — lock the theme, then focus on showing content editing.
