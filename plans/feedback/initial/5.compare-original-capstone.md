## 2. MVP SCOPE DEFINITION

### PMF Equation
```
PMF = 3-Panel Builder Mode + Listen Mode (Red Orb) + Real-Time AISP Specs
(all three required — none alone is sufficient)
```

### The 6 "Aha Moments" (Capstone Presentation Targets)
1. **The Shell:** Toggle from DRAFT to EXPERT and watch the UI instantly tighten to Enterprise Density.
2. **The Loop:** Change a hex code in the JSON and see the Reality canvas update in <100ms.
3. **The Orb:** Click "Listen", the left panel goes dark, the Red Orb pulses, and the Reality canvas stays lit.
4. **The Spec Sync:** Hover over a button in Reality, and watch the AISP Spec in the right panel auto-scroll and highlight the exact line of code.
5. **The Pipeline:** Watch the WORKFLOW tab tick through the LLM steps (Voice -> Intent -> AISP -> Render) in real-time.
6. **The Export:** Copy the AISP spec, paste it into Claude Code, and watch a production site scaffold instantly.

**MVP = Level 1 (Core Builder) + Level 2 (Site Builder) + Level 3 (Spec Engine).**

## 5. TECHNICAL QUALITY SCORECARD

| TQ-04 | **CSS Firewall** | Tailwind & `shadcn/ui` ONLY. No custom `.css` files. No `framer-motion` page transitions. | NOT STARTED |
| TQ-05 | **DDD Import Firewall** | `builder` context cannot import from `intelligence`. Use Zustand as the bridge. | NOT STARTED |
| TQ-06 | **Storage Abstraction** | Use `IProjectRepository`. Direct `localStorage.getItem()` in React is forbidden. | NOT STARTED |
