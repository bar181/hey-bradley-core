# Phase 13: Advanced Features + Content Expansion

**Prerequisite:** Phase 12 CLOSED  
**Goal:** Expand content capabilities and add power-user features before LLM integration.

## Core Deliverables
- 10+ new industry-specific examples (leveraging site context system from P12)
- Blog article editor with markdown support
- Image gallery enhancements (upload simulation, drag-to-reorder)
- Custom section builder (user-defined section types)
- Multi-page support (multi-route site configs)
- Template marketplace UI (browse/preview/apply theme+example combos)
- Export ZIP (download complete site as standalone package)
- Keyboard shortcuts for power-user workflow
- Accessibility (a11y) audit and fixes
- 100+ tests target

## Stop Point
When the next feature requires an LLM call or database → pause and plan Stage 3.
