📝 Blog Section Type SHIPPED

ADR-034 introduces blog as the 16th section type with a dedicated data model for articles (title, excerpt, author, date, tags, featured image) and 4 template variants:

VariantComponentLayout
Card GridBlogCardGrid.tsx3-column responsive grid of article cards with featured images
List ExcerptsBlogListExcerpts.tsxVertical list with thumbnail left, text right
Featured GridBlogFeaturedGrid.tsxLarge featured article at top, 2-column grid below
MinimalBlogMinimal.tsxText-only, typography-focused, no images

Blog Editor (BlogSectionSimple)

  • Layout variant selector — switch between 4 variants in real-time
  • Article management — add/remove articles with title, excerpt, author inputs
  • ImagePicker integration — featured image selection from 300+ media library
  • Toggle controls — show/hide dates and tags per section

📄 Multi-Page Architecture SHIPPED

ADR-035 adds an optional pages[] array to MasterConfig. Each page has its own id, title, slug, isHome flag, and independent sections array. Existing single-page configs work unchanged.

Architecture

  • PageConfig schema — { id, title, slug, isHome, sections[] }
  • configStore methods — addPage, removePage, reorderPages, enableMultiPage
  • Hash routing — #/about, #/services without React Router
  • MultiPageNav component — theme-colored navigation bar in preview
  • Backward compatible — configs without pages render as single-page

Spec Integration

  • Build Plan generator outputs per-page specs when multi-page is enabled
  • Page management UI in left panel with "Enable Multi-Page" button
  • Each page maintains independent section ordering

📦 ZIP Export SHIPPED

ADR-036 adds one-click project export. The exportProjectAsZip() utility uses JSZip for browser-side generation — no server required, works fully offline.

Export Contents

  • specs/north-star.md — North Star vision document
  • specs/architecture.md — SADD architecture spec
  • specs/build-plan.md — Build plan with per-page output
  • specs/features.md — Feature breakdown
  • specs/human-spec.md — Human-readable spec
  • specs/aisp.aisp — Machine-readable AISP Crystal Atom
  • config.json — Full MasterConfig
  • README.md — Project overview

Export button is located in the TopBar with a download icon. Downloads trigger via the browser Blob API.
